1) Извлеч ВСЕ файлы из архива
2) Содержимое каталога \\Neuro\Neuro\bin\Release\
        data - Содержит наюор картинок на которой будет производится обучение сети
        Neuro.exe- Исполняемый файл программы
        Все остальное мусор генерируемый Visual Studio при линковке
3) Содержимое каталога \\Neuro\Neuro\
        bin - Содержит релиз и дебаг версию откомпилированного екзешника
        obj - используется для хранения временных файлов при создании готовой программы.
        Properties - Настройки проекта
        Form1.cs - Форма с кнопками итд точнее функции реагирующие на события при нажалити итд кнопок (обработчик вьюшки)
        Form1.Designer.cs - Визуальная часть формы (вьюшка)
        Interface - Интерфейсы ядра нейросети
        InterfaceImpl - Реализация интерфейсов ядра нейросети
4) Содержимое \\Neuro\Neuro\Interface
        INeuron.cs - Интерфейс нейрона
        IPerceptron.cs - Интерфейс персептрона
        ITeacher.cs - Интерфейс обучающей нейросеть программы подмодуля
5) Содержимое \\Neuro\Neuro\InterfaceImpl
        Neuron.cs - Реализация интерфейса нейрона
        Perceptron.cs - Реализация интерфейса персептрона
        Teacher.cs - Реализация интерфейса обучающей нейросеть программы подмодуля
